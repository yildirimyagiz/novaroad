DCE GUARD (Dead Code Elimination Prevention)
────────────────────────────────────────────────────────────────────────

Problem: Optimizer may eliminate "dead" computations
Solution: Sink all results into checksum → exit code

Implementation:
  var checksum = sum1 + sum2 + sum3 + sum4;
  check checksum > 0.0 {
      yield 0;  // Forces optimizer to keep all computations
  }

Benefits:
  ✅ Prevents loop elimination
  ✅ Ensures accurate measurements
  ✅ Minimal overhead
  ✅ Industry standard practice
