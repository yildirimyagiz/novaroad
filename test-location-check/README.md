# test-location-check

A Nova application.

## Getting Started

```bash
# Build the project
nova build

# Run the project
nova run

# Start REPL
nova repl
```

## Project Structure

- `src/main.zn` - Main entry point
- `nova.toml` - Project configuration
- `tests/` - Test files
